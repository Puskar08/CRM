Welcome to the Demo

This demo shows a sidebar that overlays on top of the main content on mobile devices without displacing it.

Resize your browser window to see the mobile behavior, or use the toggle button in the top right corner.

How It Works

On desktop screens, the sidebar appears as a normal column to the left of the main content.

On mobile screens (width ≤ 768px), the sidebar:

  • Is hidden by default
  • Slides in from the left when the menu button is tapped
  • Overlays on top of the main content without displacing it
  • Can be closed by tapping the X button or the overlay
